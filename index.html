Полное руководство по игре

🎮 Что такое Mech:anic?

Mech:anic — это симулятор электронных схем. Создавайте свои логические цепи из виртуальных компонентов!

🛠️ Инструменты

✋ Рука — вкл/выкл элементы
🧤 Перчатка — перетаскивание
🔧 Гаечный ключ — провода
🪛 Отвертка — настройка (включая Коллизию)
🎨 Кисть — окрашивание
🧹 Ластик — удаление
➕ Спавн — добавление

⚡ Дающие поток

  • 📡 Сенсор — работает пока зажат
  • ⏺ Кнопка — работает пока зажата
  • 🔘 Рычаг — вкл/выкл по клику
  • 🔋 Аккумулятор — вкл/выкл по клику
  • ⚡ Генератор — вкл/выкл по клику

🔧 Активируемые

  • 💡 Лампа — загорается при сигнале
  • 🖥️ Дисплей — меняет цвет при сигнале
  • 🔲 Чип — 50+ моделей логики
  • 📏 Резистор — пропускает сигнал
  • ⚙️ Мотор — включается при сигнале
  • 🔊 Динамик — играет ноту или музыку
  • ⏱️ Таймер — задерживает сигнал

🛡️ Коллизия

У всех элементов изначально включена коллизия — они могут сталкиваться друг с другом.

Отверткой можно отключить коллизию для любого элемента.
